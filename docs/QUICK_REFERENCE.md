# Ration Curation Quick Reference

## Daily Workflow

### 1. Personal Discovery (While Browsing)
```
See good content → Share → Ration Share Extension → Auto-saved
```

### 2. Bulk Curation Session (15-30 min)
```
1. Open dashboard
2. Pick a topic/category that needs content
3. Search platforms for that topic
4. Copy 10-20 good URLs
5. Paste in Curate tab
6. Fetch → Categorize → Save
```

### 3. Outside-Your-Bubble Discovery
```
1. Go to Discover tab
2. Enter topic you don't follow (e.g., "chess")
3. Search → Browse results → Copy URLs
4. Curate tab → Save
```

## URL Cheat Sheet

### TikTok
```
Full video:     https://www.tiktok.com/@user/video/123456
Short link:     https://vm.tiktok.com/ZMhXYZ/
Profile:        https://www.tiktok.com/@user
```

### Instagram
```
Reel:           https://www.instagram.com/reel/ABC123/
Post:           https://www.instagram.com/p/ABC123/
Profile:        https://www.instagram.com/username/
```

### YouTube
```
Video:          https://www.youtube.com/watch?v=ABC123
Short link:     https://youtu.be/ABC123
Short:          https://www.youtube.com/shorts/ABC123
Channel:        https://www.youtube.com/@channelname
```

## Quality Checklist

✅ Add if:
- Teaches something useful
- High production quality
- Consistent creator
- Positive/constructive tone
- Genuinely entertaining (not rage-bait)

❌ Skip if:
- Drama/outrage content
- Clickbait titles
- Low effort reposts
- Manipulative tactics
- Controversial hot takes

## Keyboard Shortcuts (Dashboard)

- `Ctrl/Cmd + V` — Paste URLs
- `Enter` in URL box — Fetch metadata
- `Tab` through results — Navigate
- `Ctrl/Cmd + S` — Save all
